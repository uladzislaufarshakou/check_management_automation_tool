# Check Management Automation Tool

## 📌 Description
A web UI tool for mentors and their trainees.

## 📌 Features
- Pre-check: Trainee should pass pre-check quiz before he can book a check with mentor. This should save mentors' time on useless checks: mentor checks trainee's knowledge only when the quiz passed.
- Integration with Google services: The checks are synced across the Google Calendar, all books are consistent everywhere.
- Intelligent booking system: Mentor passes his free time slots in, the system counts and suggests separated slots to trainees and vise-versa.
- Smart check suggestions: The system suggests questions that should be asked on the check based on previous results and pre-checks.
- Other feature may and probably will be added there later.
